# Dora Teknoloji Web Sitesi

Modern, SEO uyumlu ve yüksek performanslı Next.js 15.4.4 ile geliştirilmiş Dora Teknoloji kurumsal web sitesi.

## 🚀 Teknolojiler

- **Next.js 15.4.4** - React framework
- **React 19** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS 4** - Styling
- **Turbopack** - Hızlı geliştirme

## 🎨 Özellikler

- ✅ SEO optimize edilmiş (Open Graph, Twitter Cards, meta tags)
- ✅ Responsive tasarım (mobil, tablet, desktop)
- ✅ Modern ve temiz arayüz
- ✅ Yüksek performans
- ✅ Smooth scroll ve animasyonlar
- ✅ Erişilebilirlik standartları

## 📦 Kurulum

```bash
npm install
```

## 🔧 Geliştirme

Development sunucusunu başlatmak için:

```bash
npm run dev
```

Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açın.

## 🏗️ Production Build

```bash
npm run build
npm start
```

## 📁 Proje Yapısı

```
doraweb/
├── app/
│   ├── layout.tsx          # Root layout (SEO meta tags)
│   ├── page.tsx             # Ana sayfa
│   └── globals.css          # Global stiller
├── components/
│   ├── Header.tsx           # Sabit header (transparent/fixed)
│   ├── Hero.tsx             # Hero section
│   ├── Solutions.tsx        # Çözümler section
│   ├── Stats.tsx            # Rakamlar section (counter animasyonu)
│   ├── WhyUs.tsx            # Neden Dora section
│   ├── Testimonials.tsx     # Müşteri deneyimleri
│   ├── Partners.tsx         # Çözüm ortakları
│   ├── CTA.tsx              # Call-to-action
│   └── Footer.tsx           # Footer
├── public/
│   └── images/              # Logo ve görseller
└── package.json
```
